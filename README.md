Тестовое задание для Effective Mobile.

Использовал рекомендуемый стек технологий, но для верстки использовал Jetpack Compose.

Стек:
1. Kotlin 
2. Coroutines
3. Flow
4. Dagger Hilt
5. MVVM
6. Jetpack Compose
7. Clean Architecture
8. Многомодульность
9. Room
10. Retrofit
